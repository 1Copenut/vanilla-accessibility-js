Accessible breadcrumbs

I know, another breadcrumb example. This one is my attempt to meet the following criteria:

  1. All breadcrumbs are valid links
  2. The current page aria-current="page" is not clickable
    • The terminal breadcrumb shows a standard cursor
    • The terminal breadcrumb's click events can be prevented with JavaScript or CSS
  3. Screen readers traverse each link individually

This last point about screen readers traversing links individually is interesting. I discovered that my unordered list was being treated as a tree view—not a standard list—when I first coded it. After some exploration, I discovered by setting the UL tag's display to "flex", each list item could be reached using the DOWN_ARROW key.
